DOMINO's Article: Data and Software Repository

This page contains the software and the datasets used to accomplish the experiments presented in the paper "Discovering Relaxed Functional Dependencies based on Multi-attribute Dominance", by Loredana Caruccio, Vincenzo Deufemia, Felix Naumann, and Giuseppe Polese, submitted for publication to IEEE TRANSACTIONS ON KNOWLEDGE AND DATA ENGINEERING. In particular, this page contains the DOMINO algorithm and the re-implemented versions of DD discovery algorithms presented by Song et al. in [1].

This link allows to download a compressed archive (zip format) containing the following files:

  • a jar file concerning the module for preparing the input for DOMINO and DD's algorithms (CreateMatrix.jar)
  • a jar file concerning the implementation of DOMINO (Domino.jar)
  • a jar file concerning the re-implementations of DD discovery algorithms (DD.jar)
  • the datasets used for the comparison
  • a readme file explaining how to use the software for replicating the experiments (Readme.txt).

[1] S. Song and L. Chen, “Differential dependencies: Reasoning and discovery,” ACM Transactions on Database Systems (TODS), vol. 36, pp. 16:1–16:41, 2011.
